# Form 5500

## Overview

Form 5500 is an annual report filed with the Department of Labor for employee benefit plans subject to ERISA. It contains key information about plan sponsors, including their Employer Identification Number (EIN).

## Table Schema

| Column | Type | Key | Description |
|--------|------|-----|-------------|
| `id` | INT | PK | Primary key |
| `ack_id` | VARCHAR | UK | DOL acknowledgment ID |
| `company_unique_id` | TEXT | FK | Linked company (post-backfill) |
| `sponsor_dfe_ein` | VARCHAR | | Employer EIN |
| `sponsor_dfe_name` | VARCHAR | | Company name |
| `spons_dfe_mail_us_state` | VARCHAR | | State code |
| `form_year` | INT | | Filing year |
| `tot_active_partcp_cnt` | INT | | Active participants |
| `plan_name` | VARCHAR | | Benefit plan name |
| `raw_payload` | JSONB | | Full filing data |
