Git Bash (MINGW64) is a Unix-like shell for Windows, but:
- It has its own PATH separate from Windows PATH
- Docker Desktop adds "docker" to Windows PATH
- But NOT to Git Bash PATH automatically

This is normal and expected behavior!

Solution: Use PowerShell or CMD for Docker commands
          OR add Docker to Git Bash PATH (advanced)
